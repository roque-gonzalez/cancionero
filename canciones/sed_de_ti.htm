SED DE TI

1. Sed de ti para vivir, sed de ti para buscarte. Dame sed para sentirme viva en ti, dame sed de ti.

2. Sed de ti para esperar, sed de ti para quererte. Dame sed para aprender a caminar, dame sed de ti.

3. Sed de ti para escuchar, sed de ti para sanarme. Dame sed para empezar a respirar, dame sed de ti.

4. Sed de ti para mirar, sed de ti para guiarme. Dame sed para volver a comenzar, dame sed de ti, dame sed de ti.

(Salomé Arricibita)
